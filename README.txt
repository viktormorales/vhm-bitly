== Installation ==

1. Upload `vhm-bitly.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a Generic Access Token on Bitly
1. Paste the token in the plugins settings page.
